# Kayñiawlu — rédaction mémoire

Documents de rédaction du mémoire de fin d'études (ISI Master 2 GL).

## Mémoire généré

Fichier principal : **`Conception et réalisation d'une solution SaaS de gestion pour les couturiers du Sénégal - Kayniawlu.docx`**

### Personnalisation rapide

1. Ouvrir le `.docx` dans **Microsoft Word** ou **LibreOffice**.
2. Remplacer les placeholders `[VOTRE PRÉNOM ET NOM]`, `[NOM DE VOTRE ENCADREUR]`, etc. (page de garde, dédicace).
3. Clic droit sur le **Sommaire** → **Mettre à jour les champs** → **Mettre à jour toute la table**.
4. Insérer les figures BPMN/UML marquées `[À insérer]` (Draw.io, captures d'écran du projet).

### Régénération depuis le script

Modifier les constantes en tête de `generate_memoire.py`, puis :

```bash
cd redaction && python3 generate_memoire.py
```
